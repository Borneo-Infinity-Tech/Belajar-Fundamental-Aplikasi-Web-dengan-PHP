# Praktikum Menggabungkan PHP dengan HTML dan CSS

## Praktikum 1 - Biodata Siswa

## Tujuan

Menampilkan data dari variabel PHP menggunakan elemen HTML.

## Ketentuan

Buatlah sebuah halaman web dengan ketentuan berikut.

Judul halaman:

```
Biodata Siswa
```

Simpan data berikut ke dalam variabel PHP.

<table><tbody><tr><td><strong>Variabel</strong></td><td><strong>Nilai</strong></td></tr><tr><td>Nama</td><td>Tuliskan Nama Kalian</td></tr><tr><td>Kelas</td><td>Tuliskan Kelas Kalian</td></tr><tr><td>Jurusan</td><td>Pengembangan Perangkat Lunak dan Gim</td></tr><tr><td>Sekolah</td><td>SMK ISFI Banjarmasin</td></tr></tbody></table>

Tampilkan data tersebut menggunakan:

- Heading `<h1>`
- List `<ul>`

Sehingga informasi ditampilkan dua kali, yaitu dalam bentuk **list** dan **table**.

## Praktikum 2 - Data Mata Pelajaran

## Tujuan

Menggunakan beberapa variabel PHP untuk membentuk isi halaman.

## Ketentuan

Buat halaman web dengan judul

```
Daftar Mata Pelajaran
```

Simpan data berikut ke dalam variabel PHP.

- Bahasa Indonesia
- Matematika
- Pemrograman Web
- Basis Data
- Pendidikan Pancasila

Kemudian tampilkan:

1.  Heading
2.  Tabel dengan kolom:

<table><tbody><tr><td>No</td><td>Mata Pelajaran</td></tr><tr><td>Nomor harus ditulis manual (1–5).</td><td>&nbsp;</td></tr></tbody></table>

## Praktikum 3 - Data Nilai Siswa

## Tujuan

Menampilkan beberapa variabel ke dalam tabel HTML.

## Ketentuan

Buat halaman web berjudul

```
Data Nilai Siswa
```

Simpan data berikut ke dalam variabel PHP berbentuk array asosiatif.

| Nama  | Nilai |
| ----- | ----- |
| Andi  | 85    |
| Budi  | 90    |
| Citra | 88    |

Kemudian tampilkan:

- Judul menggunakan `<h1>`
- Tabel dengan kolom:

| Nama | Nilai |

Tambahkan satu paragraf di bawah tabel:

```
Jumlah siswa : 3 orang
```

Jumlah siswa berasal dari variabel PHP.

## Praktikum 4 - Profil Sekolah

## Tujuan

Menggabungkan teks PHP dengan HTML yang lebih lengkap.

## Ketentuan

Buat halaman web berjudul

```
Profil Sekolah
```

Simpan data berikut ke dalam variabel PHP.

- Nama Sekolah
- Alamat
- Kepala Sekolah
- Tahun Berdiri
- Jumlah Guru

Kemudian tampilkan:

### Bagian 1

Heading

```
SMK ISFI Banjarmasin
```

(nama berasal dari variabel PHP)

### Bagian 2

List berisi:

- Alamat
- Kepala Sekolah
- Tahun Berdiri
- Jumlah Guru

### Bagian 3

Informasi daftar jurusan dalam bentuk tabel.

Tabel

| Informasi | Keterangan |

Semua data berasal dari variabel PHP.

## Praktikum 5 - Dashboard Informasi Siswa

## Tujuan

Menggabungkan heading, list, tabel, CSS sederhana, dan data PHP dalam satu halaman web.

## Ketentuan

Buat sebuah halaman dashboard sederhana dengan tampilan yang rapi menggunakan CSS.

### Bagian Header

Menampilkan judul

```
Dashboard Informasi Siswa
```

Judul berasal dari PHP.

### Bagian Informasi

Simpan data berikut pada variabel PHP.

- Nama
- NIS
- Kelas
- Jurusan
- Alamat

Kemudian tampilkan dalam bentuk list.
